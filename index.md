## 📖 项目简介

SimPhish 是一个专为企业内部安全意识培训设计的钓鱼邮件模拟平台。通过模拟真实的钓鱼攻击场景，帮助组织提高员工的安全意识，识别和防范网络钓鱼威胁。

### 🚀 核心价值

- **教育导向**：提升员工安全意识。
- **安全可控**：确保测试环境安全。
- **数据追踪**：记录员工行为数据。
- **灵活定制**：自定义邮件模板和钓鱼页面。

---

## 🛠️ 功能特性

### 📧 邮件管理
- **模板编辑**：支持 HTML 和富文本编辑器。
- **变量替换**：内置多种变量（邮箱、姓名等）。
- **附件支持**：可添加最多 10 个附件。
- **SMTP 配置**：支持 SSL/TLS 和 STARTTLS 加密。
- **发送控制**：支持并发控制和速率限制。

![邮件管理功能](https://github.com/user-attachments/assets/22d4c61f-b22e-4167-b1ee-12b67c7565de)

### 🌐 钓鱼页面
- **页面编辑**：支持自定义 HTML 或 URL 导入。
- **变量支持**：页面内容支持变量替换。
- **服务器配置**：可配置钓鱼服务器地址和端口。
- **重定向**：支持提交后重定向到指定 URL。

## ⚙️ 配置说明

在首次运行 SimPhish 之前，请完成以下配置：

### 1. 设置环境
确保您的计算机运行在以下平台：
- Windows 10 或更高版本
- MacOS Catalina 或更高版本

### 2. 配置 SMTP
- 根据您的需求填写 SMTP 服务器地址。
- 选择加密方式（如 SSL/TLS 或 STARTTLS）。

### 3. 配置钓鱼页面
- 编辑您希望发送的钓鱼邮件模板。
- 确保重定向链接准确无误。
